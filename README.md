# UPI Impulse Trap Analysis

This project analyzes impulsive UPI spending behavior and financial regret among Indian college students using:

- Exploratory Data Analysis (EDA)
- Natural Language Processing (NLP)
- Machine Learning (Clustering & Classification)

## Dataset
Primary data collected via Google Form (105 responses)

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Scikit-learn
- NLTK / TextBlob
- Power BI

## Objective
To identify behavioral triggers of impulsive spending and predict financial regret patterns.
